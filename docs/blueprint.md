# **App Name**: ShieldCheck AI

## Core Features:

- Text Analysis & Scam Detection: Users can paste suspicious text (from messages, emails, etc.). The AI tool analyzes it for scam indicators, provides a risk score, and classifies the potential scam type.
- Link Verification Tool: Users input suspicious URLs. The AI tool inspects them for phishing signs, unusual domains, URL shortener redirects, and other fraudulent patterns to assess risk.
- Screenshot Scan with OCR: Allows users to upload a screenshot. The app uses Optical Character Recognition (OCR) to extract text from the image, which is then fed to the AI tool for scam analysis.
- Dynamic Risk Assessment & Explanation: The AI tool generates a clear risk assessment (e.g., 'Low' to 'Very High'), provides plain-language explanations of detected suspicious elements, and lists reasons for the assessment.
- Actionable Safety Recommendations: Based on the AI tool's analysis, the app provides concrete, easy-to-understand recommendations on immediate actions the user should take (e.g., 'Do not click,' 'Block sender').
- Analysis History (Local & Basic): A basic, local-only log of recent analyses (e.g., last 5-10 entries), including the date, identified risk level, and type of threat. No cloud sync for MVP to keep costs low.
- Educational Mini-Guides: A section offering short, accessible, pre-defined guides on common scam types and general digital safety tips to empower users with foundational knowledge.

## Style Guidelines:

- Primary Color: Trustworthy Blue (#1973E1). This clear, medium-saturation blue instills confidence and readability for critical information, reflecting the app's cybersecurity purpose.
- Background Color: Serene Light Blue (#E3EDF5). A heavily desaturated, light blue from the same hue family as the primary color provides a clean, calming canvas, minimizing eye strain.
- Accent Color: Vibrant Cyan (#66DBDB). An analogous color chosen for contrast and energy, used sparingly for calls to action, highlights, and urgent alerts, ensuring they stand out without overwhelming the user.
- Headline and Body Font: 'Inter' (sans-serif). A highly legible, modern sans-serif typeface, 'Inter' offers excellent readability across all text sizes, fitting a simple and reliable user experience for diverse age groups.
- Use a set of simple, line-art style icons. Icons should be easily understandable and communicate their function instantly, supporting clear navigation and accessibility for users of all digital literacy levels.
- A clean, spacious, and mobile-first layout prioritizing readability and ease of interaction. Key buttons are large and clearly labeled. Content blocks are well-separated, reducing visual clutter for an approachable interface.
- Implement subtle, fluid animations for state changes and user feedback. These transitions should feel natural and assist user comprehension without causing distraction or delays, reinforcing a premium, responsive feel.
